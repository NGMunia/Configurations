
This configuration seeks to achieve the following:
    - DMVPN with IPsec 
    - OSPF routing between devices and DMVPN network
    - Configure QoS to:
         - restrict bandwidth social media sites (CS1)
         - block torrent sites.
         - Allow default traffic (CS0)
    - Automate TFTP backup of all devices' configuration every weekday at 09:30 AM using EEM
      on server IP 192.168.255.254
      
    - Collect/analyze data traffic using:
         - Syslog
         - SNMP
         - NetFlow
         - Syslog
    - Device security features using
         - CoPP
    - IP services:
         - NTP
